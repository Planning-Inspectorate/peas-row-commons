# Utils
- This is the directory that holds all util modules.

## Search
- This is a util for building out `WHERE` clauses for generic search functionalities.
- It is worth noting that the more things you add to search across, the slower that search will be, so there should be care to make sure you have optimised any searches & joins, and have laid the groundwork in the database first (e.g. by adding the exact indexing needed) before adding new items to search across
